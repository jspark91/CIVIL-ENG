# CIVIL-ENG

DXF alignment map viewer tools for civil engineering workflows.

This package converts AutoCAD DXF alignment linework in Korean GRS80 coordinate systems to WGS84 GeoJSON and interactive map HTML previews for:

- NAVER Map
- Kakao Map
- Google Maps

The browser UI separates generated outputs by provider:

- `outputs/naver_map`
- `outputs/kakao_map`
- `outputs/google_map`

Start with `docs/naver-map-dxf.md` for usage, API key setup, roadview/street-view notes, and portable packaging.
